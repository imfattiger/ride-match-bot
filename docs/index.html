要收費嗎？
平台媒合完全免費，不收手續費。司機與乘客之間的油資分攤由雙方自行協議。
共乘合法嗎？
合法。依交通部函釋，分攤油資、過路費的共乘行為不屬於汽車運輸業。平台媒合不經手車資。
配對後怎麼聯絡？
配對成功後系統推播通知，並提供對方 LINE ID，由雙方自行聯絡細節。
